# InSAR Damage Assessment

This project maps building and infrastructure damage over a user-defined area of interest using Sentinel-1 SAR interferometric coherence loss as a proxy: collapsed or heavily damaged structures decorrelate the radar return between passes, while intact, unchanged ground mostly does not. The pipeline queries the Copernicus Data Space Ecosystem catalog for Sentinel-1 SLC acquisitions over the AOI and automatically selects a triplet of scenes on a shared orbital track — two pre-event images and one post-event image — then downloads and processes them with ESA SNAP into two coherence products: a "baseline" pair (pre-event vs. pre-event, characterizing ordinary non-damage decorrelation from vegetation, weather, and seasonal change) and a "co-event" pair (pre-event vs. post-event, capturing whatever additionally decorrelated during the event window). Pixels where coherence drops significantly more in the co-event pair than in the baseline pair are flagged as damage candidates, connectivity-filtered to suppress speckle noise, and the result is written out as a georeferenced multi-band GeoTIFF, a static comparison figure, an interactive HTML map with toggleable coherence/damage layers, and a JSON summary of affected-area statistics — giving both a quick visual read and machine-readable numbers for any AOI and date range supplied.
